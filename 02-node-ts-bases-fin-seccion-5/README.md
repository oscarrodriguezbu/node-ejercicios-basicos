# 02 Bases de Node


Correr en Dev
```
npm run dev
```

Las pruebas se van a hacer sobre las funcionalidades de los archivos, por eso se comenta el app.ts

Correr pruebas:
```
npm run test
```
Correr pruebas en modo observable: (se sale con q)
```
npm run test:watch
```

Correr pruebas con cobertura:
```
npm run test:coverage
```
